HCL Syntax Rules

  • Blocks: Containers for content.
    resource "aws_instance" "web" { ... }
  • Arguments: Assign a value to a name.
    ami = "ami-123456"
  • Expressions: Represent values, literally or referenced.
    var.instance_type
